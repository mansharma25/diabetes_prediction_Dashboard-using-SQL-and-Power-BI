# Patient Health Analytics Dashboard

This project demonstrates the creation of an **interactive Power BI dashboard** for analyzing patient health data. The dataset was sourced from Kaggle, processed using SQL Server, and visualized in Power BI.

The dataset contains the following fields:
- `gender`: Gender of the patient.
- `age`: Age of the patient.
- `hypertension`: Binary indicator (0 or 1) for hypertension.
- `heart_disease`: Binary indicator (0 or 1) for heart disease.
- `smoking_history`: Categorized smoking habits (e.g., never, former, current).
- `bmi`: Body Mass Index (BMI).
- `HbA1c_level`: Hemoglobin A1c levels.
- `blood_glucose_level`: Blood glucose levels.
- `diabetes`: Binary indicator (0 or 1) for diabetes.
